# dateparsejs

package for parsing date

### Getting Started

#### Install

```
$ npm install dateparsejs
```

#### Usage

```
const dateparsejs = require('dateparsejs')

dateparsejs('2018-08-05T00:00:00.000Z')
// 2018 August 05

import dateparsejs from 'dateparsejs'
dateparsejs('2018-08-05T00:00:00.000Z')
// 2018 August 05
```
